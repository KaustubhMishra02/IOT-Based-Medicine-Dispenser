<H1 align="center">Medicine Dispensing System</H1>
<p align="center">Flask webapp for an IOT based medicine dispenser using RaspberryPi and Python.</p>
<p align="center">Concepts required: Unerstanding of RaspberryPi GPIO, Flask Web-Framework, Python </p>
<p align="center"> Python v3.10.4 || Flask v2.2.2 || Raspbian OS v5.15

## Components
- RaspberryPi
- Servomotors
- Stepper motors
- Buzzer module
- 7 segment circular medicine container
- Breadboard and jumper wires

## Working and Deployment
- Boot up a RaspberryPi.
- Clone the repo and copy it to the RaspberryPi.
- Make the connections as specified in the project report.
- Open a terminal on the RPi and execute prj.py executable file.
- Open the specified host address on a browser.
- Specify the day for which medicine has to be dispensed.
- That week's data can be accessed by going to host:url/status.
